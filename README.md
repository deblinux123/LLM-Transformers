# 🤗 Learn Large Language Models (LLMs) with Hugging Face Transformers

Welcome to this repository!

This repository is a practical companion for learning **Large Language Models (LLMs)** using the **Hugging Face Transformers** ecosystem. It is designed to help you understand the core concepts behind modern AI models while gaining hands-on experience through Python examples and exercises.

Whether you're just getting started with LLMs or looking to strengthen your knowledge of the Hugging Face ecosystem, this repository provides a structured learning path from the fundamentals to more advanced topics.

---

## 📖 What You'll Learn

Throughout this repository, you'll explore topics including:

* Introduction to Large Language Models
* The Hugging Face ecosystem
* Transformers library fundamentals
* Loading pre-trained models
* Working with tokenizers
* Using Pipelines for common NLP tasks
* Text generation
* Question answering
* Summarization
* Translation
* Text classification
* Feature extraction and embeddings
* Fine-tuning transformer models
* Parameter-efficient fine-tuning (PEFT)
* Inference optimization
* Building practical LLM applications

Each chapter focuses on explaining the concepts first, followed by clear Python examples that you can run and modify yourself.

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/your-username/your-repository.git
```

### Move into the Project Directory

```bash
cd your-repository
```

### Create a Virtual Environment (Recommended)

**Linux / macOS**

```bash
python -m venv .venv
source .venv/bin/activate
```

**Windows**

```powershell
python -m venv .venv
.venv\Scripts\activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```
